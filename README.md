Number Guessing Game 🎯

A command-line Number Guessing Game built using Python. The player selects a difficulty level, and the program generates a random number within a specific range. The goal is to guess the correct number within the given number of attempts.

Features
Three difficulty levels:
Easy (1–100)
Medium (1–200)
Hard (1–500)
Limited attempts based on difficulty
Hints after each guess ("Try higher!" or "Try lower!")
Win/Lose detection
Option to play multiple rounds
Technologies Used
Python
Random Module
How to Run
Install Python on your system.
Download or clone this repository.
Open a terminal in the project folder.
Run the program:
python number_guessing_game.py
How to Play
Choose a difficulty level.
Enter your guesses.
Use the hints to get closer to the correct number.
Guess the number before running out of attempts.
Choose whether to play another round.
